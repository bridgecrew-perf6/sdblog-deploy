Welcome to Rocket11ty!

It's a decent blogging platform made with Halfmoon and 11ty.

  • Change the values in _data/metadata.json to your values.
  • If you are hosting your site on Netlify and want to enable extra features like the contact form and NetlifyCMS, enable Git Gateway, Netlify Identity, and Netlify Forms (should already enabled.)
  • Delete this message in includes/layouts/base.njk from Start deleting! to Stop deleting! Happy Blogging!
